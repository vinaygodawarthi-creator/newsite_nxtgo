## How to Use the Tracking Modal

1. **Open Modal**: Click "Track Shipment" button in the header
2. **Enter Number**: Type an 8-digit tracking number
3. **Submit**: Click "Track Shipment" button
4. **Close Modal**: 
   - Click the X button
   - Click the overlay background
   - Press Escape key
